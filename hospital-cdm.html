Hospital CDM Inventory

Complete inventory of hospital pricing data availability

Under CMS Hospital Price Transparency rules, hospitals must publish charge master (CDM) files. But compliance is inconsistent—some hospitals publish, others don't. We've profiled every hospital in the US to tell you: Who has data available? Where is it hosted? What format is it in? What's the file size? We test quarterly and track changes over time.

Data Schema

Table: hospital_cdm_inventory

Columns:
- hospital_id          : Unique identifier
- hospital_name        : Facility name
- city                 : City location  
- state                : Two-letter state code
- state_name           : Full state name
- working_domain       : Website domain
- cms_hpt_found        : Boolean (TRUE if file exists)
- cms_hpt_url          : Full URL to CMS-HPT.txt file
- num_locations        : Number of facility locations
- cdm_files_accessible : Count of downloadable CDM files
- total_size_mb        : Total data size in MB
- file_formats         : CSV, JSON, Excel, etc.
- status               : SUCCESS | PARTIAL | NOT_FOUND
- last_checked         : Timestamp of last validation

Example Query:
SELECT hospital_name, city, total_size_mb, cdm_files_accessible
FROM hospital_cdm_inventory
WHERE state = 'CA' 
  AND status = 'SUCCESS'
ORDER BY total_size_mb DESC
LIMIT 20;
