Payer Rate Transparency Platform

Every negotiated rate from 900+ US health insurers

Under the CMS Transparency in Coverage rule, health insurers must publish negotiated rates with providers. We aggregate all these files, normalize the data, and deliver it via a clean REST API. You get every negotiated rate: what UnitedHealthcare pays Hospital A for procedure X, what Aetna pays Hospital B, what regional payers negotiate—all in one queryable database.

Starting at $10,000/year

What You Get

  • 50M+ negotiated rates from 900+ insurance companies
  • REST API with complete documentation and code samples
  • Monthly data refreshes — we monitor payer file changes automatically
  • White-label licensing — remove our name entirely, build your branded tool
  • Clean, normalized data — we handle parsing nested JSON, standardizing procedure codes, resolving duplicates
